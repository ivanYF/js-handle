# js 小功能 模块集合

> 提供一些简易的js模块操作


## 介绍

- 利用canvas 制作 画笔的功能 

- dom 模块拖拽 获取在页面中的偏移位置，限制移动的边界。

- 无js库依赖的lazyload js module


## 将项目克隆到本地

```bash
# 定位到任意目录
$ cd path/to/root

# 克隆仓库到指定的文件夹
$ git clone https://github.com/ivanYF/js-handle.git [project-name] --depth 1

# 进入指定的文件夹
$ cd [project-name]
```
